## Running locally

Just open `index.html` in a browser — no build step, no dependencies.

```bash
git clone https://github.com/nishanthi-builds/nishanthi-builds.github.io
cd nishanthi-builds.github.io
open index.html
```

---

## Deploying to GitHub Pages

1. Repo must be named exactly `nishanthi-builds.github.io`
2. Push `index.html` to the `main` branch
3. Go to **Settings → Pages → Source → main / root**
4. Site goes live at `https://nishanthi-builds.github.io` within a minute
